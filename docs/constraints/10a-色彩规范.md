# 10a 色彩规范（基于品牌色板）

> 落实 10-UI 的 A1(令牌)/A2(主题)/A4(配色)。本规范是**颜色的唯一来源**：
> 先定语义令牌与用法，组件只引语义令牌，禁硬编码十六进制。

---

## 1. 基础色板（原始令牌 = 你提供的）

```css
--primary-100:#3F51B5;  /* 主色(靛蓝, Indigo 500) — 主操作/品牌 */
--primary-200:#757de8;  /* 主色-亮 — hover/聚焦/装饰，不做小字背景 */
--primary-300:#e8eaf6;  /* 主色-淡(选中行/信息底) — 由原 #dedeff 调整为同色相低饱和淡靛, 见 §4.1 */
--primary-350:#dce1f5;  /* 主色-淡(深一档) — 芯片/标签底, 从淡底里浮出 */
--accent-100:#2196F3;   /* 强调(蓝) — 大色块/图标/大标题，⚠不做小字/链接 */
--accent-200:#003f8f;   /* 强调-深 — 强调文字/链接(在白底) */
--text-100:#333333;     /* 正文 */
--text-200:#5c5c5c;     /* 次要文字 */
--bg-100:#FFFFFF;       /* 表面(卡片/容器) */
--bg-200:#f5f5f5;       /* 页面底 */
--bg-300:#cccccc;       /* 分割线(细) */
```

## 2. 语义令牌映射（浅色，组件只引这一层）

| 语义令牌 | = 基础 | 用途 |
|---|---|---|
| `--color-bg` | bg-200 `#f5f5f5` | 页面底色 |
| `--color-surface` | bg-100 `#ffffff` | 卡片/容器/输入框底 |
| `--color-surface-hover` | bg-200 `#f5f5f5` | 列表/行 hover |
| `--color-border` | bg-300 `#cccccc` | 分割线(细) |
| `--color-border-strong` | **`#9e9e9e`**(补) | 输入框/控件边界(需可辨, 见 §5) |
| `--color-text` | text-100 `#333333` | 正文/标题 |
| `--color-text-secondary` | text-200 `#5c5c5c` | 次要/说明 |
| `--color-text-disabled` | **`#9e9e9e`**(补) | 禁用文字 |
| `--color-primary` | primary-100 `#3F51B5` | 主按钮/主操作/品牌 |
| `--color-primary-hover` | **`#37459b`**(深 8%) | 主按钮 hover/active |
| `--color-primary-soft` | primary-200 `#757de8` | 聚焦环/次级强调/装饰 |
| `--color-primary-subtle` | primary-300 `#e8eaf6` | 选中行/信息底高亮 |
| `--color-primary-subtle-strong` | primary-350 `#dce1f5` | Chip/Tag 底(从淡底里浮出) |
| `--color-on-primary` | `#ffffff` | 主色之上的文字 |
| `--color-accent` | accent-100 `#2196F3` | 大色块/图标/图表/大标题 |
| `--color-accent-text` | accent-200 `#003f8f` | 强调文字/链接(白底) |
| `--color-on-accent` | `#ffffff` | accent-200 之上的文字 |

## 3. 状态色（补全 · 原色板缺，A4/A10 需要）

> 你给的色板没有 成功/警告/危险/信息 色，按 Material 补全（可调）：

| 语义 | 主色 | 浅底 | 文字(在浅底) | 用途 |
|---|---|---|---|---|
| success | `#2e7d32` | `#e8f5e9` | `#1b5e20` | 成功 toast/状态 |
| warning | `#ed6c02` | `#fff4e5` | `#7a3e00` | 警告 |
| danger | `#d32f2f` | `#fdecea` | `#b71c1c` | 错误/危险动作 |
| info | accent-100 `#2196F3` | primary-350 `#dce1f5` | `#283593`(深靛) | 信息/进行中 Chip(底用 subtle-strong) |

危险动作(发布/删除)按钮：底 `danger #d32f2f` + 白字（对比 ≈5.9:1 ✓）。

## 4. 用法规则（primary 三档/accent 两档怎么用）

- **primary-100**：主按钮填充、选中标签页下划线、关键链接强调、品牌。白字 OK。
- **primary-200**：**不要做小字背景**；用于 hover/active 的次级、聚焦环(focus ring)、装饰描边、图表辅助色。
- **primary-300**：**淡底**——选中行高亮、信息横幅底、Chip/Tag 底、当前节点高亮；其上用 `text-100`。
- **accent-100(#2196F3)**：**只做大色块/图标/图表/大标题(≥18.66px 粗 或 ≥24px)**；**不要做小字背景、不要做正文链接色**(对比不足, 见 §5)。
- **accent-200(#003f8f)**：白底上的**强调文字/链接/小字强调**首选；也可作深色强调填充(白字)。

### 4.1 淡底派生规则 ★（避免"色块突兀"）
- 规则：**淡底/选中底必须是主色的"同色相、低饱和、高明度"派生**(主色 + 大比例白)，
  **不得另选一个高饱和的相邻色**——否则与整体不同色相、过艳过亮，在页面里"冲/飘"。
- 反例(原色板)：主色靛蓝 `#3F51B5`，淡底却给偏紫高饱和的 `#dedeff` → 选中行突兀。
  修正：`primary-300 = #e8eaf6`(Indigo 50，主色稀释)，融入页面只做轻提示。
- 两档：`primary-300 #e8eaf6` 做**行/横幅淡底**；`primary-350 #dce1f5` 做**芯片/标签底**(比淡底深一档以浮出)，
  其上文字用同 family 深色 `#283593`；选中项的图标/文字可用主色 `#3F51B5` 呼应。

## 5. ★ 对比度与可达性（实测，必须遵守）

> WCAG AA：正文 ≥4.5:1，大字 ≥3:1，非文本控件边界 ≥3:1。以下为实测近似值。

| 前景 / 背景 | 比值 | 结论 |
|---|---|---|
| text-100 `#333` / 白 | ≈12.6:1 | ✅ AAA 正文 |
| text-200 `#5c5c5c` / 白 | ≈6.7:1 | ✅ AA 次要文字 |
| 白 / primary-100 `#3F51B5` | ≈6.9:1 | ✅ 主按钮白字 OK |
| 白 / primary-200 `#757de8` | ≈3.6:1 | ⚠ **仅大字/装饰**，小字不达标 |
| text-100 / primary-300 `#e8eaf6` | ≈11.7:1 | ✅ 淡底+深字 OK |
| `#283593` / primary-350 `#dce1f5` | ≈7.8:1 | ✅ 芯片底+深靛字 OK |
| 白 / accent-100 `#2196F3` | ≈3.1:1 | ⚠ **小字不达标**，仅大字/图标 |
| accent-100 `#2196F3` / 白(做链接) | ≈3.1:1 | ❌ **正文链接不达标 → 用 accent-200** |
| 白 / accent-200 `#003f8f` | ≈10:1 | ✅ AAA，强调文字/链接首选 |
| bg-300 `#ccc` / 白(做边界) | ≈1.6:1 | ⚠ 仅细分割线；**控件边界用 border-strong `#9e9e9e`(≈2.6→3:1)** |

**两条硬规则**：① **正文链接/小字强调一律用 `accent-200`，不用 accent-100**；
② **白字小字一律放 primary-100/accent-200/danger 等深色之上，不放 primary-200/accent-100**。

## 6. 深色模式映射（补 · A2 要求，原色板仅浅色）

| 语义令牌 | 深色取值 | 说明 |
|---|---|---|
| `--color-bg` | `#121212` | 页面底 |
| `--color-surface` | `#1e1e24` | 卡片(略带靛味) |
| `--color-surface-hover` | `#26262e` | hover |
| `--color-border` | `#34343c` | 分割线 |
| `--color-text` | `#ececf1` | 正文 |
| `--color-text-secondary` | `#a0a0ab` | 次要 |
| `--color-primary` | primary-200 `#757de8` | **深色下用更亮的靛蓝**(在深底对比足) |
| `--color-primary-subtle` | `#2a2c52` | 淡底(暗) |
| `--color-on-primary` | `#0f1020` | 亮靛上用深字 |
| `--color-accent-text` | `#5aa9f5` | 深底上的强调蓝(提亮 accent) |

> 深色不是反色：primary 改用更亮的 #757de8，accent 文本提亮，保证深底对比。

## 7. 落地

### CSS 变量（前端 :root / [data-theme=dark]）
```css
:root{ --color-bg:#f5f5f5; --color-surface:#fff; --color-text:#333; --color-text-secondary:#5c5c5c;
  --color-primary:#3F51B5; --color-primary-hover:#37459b; --color-primary-soft:#757de8;
  --color-primary-subtle:#e8eaf6; --color-primary-subtle-strong:#dce1f5; --color-on-primary:#fff;
  --color-accent:#2196F3; --color-accent-text:#003f8f; --color-border:#ccc; --color-border-strong:#9e9e9e;
  --color-success:#2e7d32; --color-warning:#ed6c02; --color-danger:#d32f2f; }
[data-theme="dark"]{ --color-bg:#121212; --color-surface:#1e1e24; --color-text:#ececf1;
  --color-text-secondary:#a0a0ab; --color-primary:#757de8; --color-on-primary:#0f1020;
  --color-accent-text:#5aa9f5; --color-border:#34343c; }
```

### Tailwind theme（B7 · 引 CSS 变量, 自动跟主题）
```js
// tailwind.config theme.extend.colors
colors:{
  bg:'var(--color-bg)', surface:'var(--color-surface)',
  text:{ DEFAULT:'var(--color-text)', secondary:'var(--color-text-secondary)' },
  primary:{ DEFAULT:'var(--color-primary)', hover:'var(--color-primary-hover)',
            soft:'var(--color-primary-soft)', subtle:'var(--color-primary-subtle)',
            'subtle-strong':'var(--color-primary-subtle-strong)',
            fg:'var(--color-on-primary)' },
  accent:{ DEFAULT:'var(--color-accent)', text:'var(--color-accent-text)' },
  border:{ DEFAULT:'var(--color-border)', strong:'var(--color-border-strong)' },
  success:'var(--color-success)', warning:'var(--color-warning)', danger:'var(--color-danger)' }
// 用法: bg-surface text-text-secondary border-border-strong text-accent-text  (禁 text-[#2196F3])
```

### RN theme（移动端 13 · 同名导出）
同一份语义令牌导出为 `colors` 对象（light/dark），`makeStyles(colors)` 引用，禁硬编码。

## 8. Do / Don't
- ✅ 主按钮：`bg-primary text-primary-fg`，hover `bg-primary-hover`。
- ✅ 链接/强调文字：`text-accent-text`(#003f8f)。
- ✅ 选中行/信息底：`bg-primary-subtle`(#e8eaf6) + `text-text`；Chip 底用 `bg-primary-subtle-strong`(#dce1f5)。
- ✅ 淡底必须由主色稀释(同色相低饱和)，禁另选高饱和相邻色(见 §4.1)。
- ✅ 控件边界：`border-border-strong`；细分割：`border-border`。
- ❌ 白小字放 `#757de8`/`#2196F3` 上（对比不足）。
- ❌ 用 `#2196F3` 当正文链接色（改 `#003f8f`）。
- ❌ 淡底用与主色不同色相的高饱和色(如靛蓝页面配偏紫 `#dedeff` / 偏青 `#c8ffff`)。
- ❌ 组件里写死十六进制 / 用非令牌色。
