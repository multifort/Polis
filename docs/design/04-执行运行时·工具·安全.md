## 1. Agent Runtime 执行内核（无状态优先）

### 1.1 执行接口
```python
class AgentRuntime:
    async def execute(self, agent: AgentVersion, node: Node, org_id: str) -> NodeResult:
        ctx   = ContextAssembler.build(agent, node, org_id)   # §2 注入三样
        result = await self._loop(agent, node, ctx)           # §3 执行循环
        envelope = ResultEnvelope.from_(result)               # 出处+摘要(05)
        MemoryCenter.write(org_id, agent, envelope)           # 写回(05)
        return NodeResult(ok=result.ok, envelope=envelope, replannable=result.soft_fail)
```
- **无状态**：每次 execute 全新构建上下文、用完即弃；可重试/换 Worker。
- 有状态/长生命周期 Agent = V2+ 增强（需 Agent State Manager），V1 不做。

### 1.2 ★ 运行时注入三样（统一约定）
```python
class ContextAssembler:
    def build(agent, node, org_id) -> ExecCtx:
        slice  = MemoryCenter.retrieve(org_id, agent, node)   # ①上下文切片(05 RAG, 摘要)
        skills = SkillLoader.load(agent.skills, agent.authority) # ②启用技能(手册+工具)
        model  = ModelGateway.resolve(agent.model, org_id)    # ③模型(06 LiteLLM)
        cred   = CredentialBroker.scoped(org_id, model.id, node.task_id) # ③任务级短时凭证(06)
        goal   = node.acceptanceCriteria                      # 目标每步在场
        return ExecCtx(goal, slice, skills, model, cred, node)
```

## 2. 技能双形态加载

```python
class SkillLoader:
    def load(skill_refs, authority) -> Loaded:
        manuals, tools = [], []
        for ref in skill_refs:                       # name@version
            s = SkillStore.get(ref)
            if s.kind == "manual":                   # 手册型: 注入 prompt
                manuals.append(s.content)            # SKILL.md 正文
            elif s.kind == "tool":                   # 工具型: 经 MCP 暴露
                if s.tool not in authority.allowedTools:   # ★最小权限校验
                    continue
                tools.append(McpRegistry.bind(s.mcp_server, s.tool))
        return Loaded(system_append="\n".join(manuals), tools=tools)
```
- 手册讲"怎么做"(进系统提示词)，工具讲"用什么做"(MCP 函数)。手册运行时无关。

## 3. 执行循环（lite-agent，复用 LangGraph 或最小循环二选一）

```python
async def _loop(agent, node, ctx) -> Raw:
    msgs = [system(agent.prompt + ctx.skills.system_append + ctx.goal),
            user(node.inputHint + ctx.slice)]
    for _ in range(MAX_STEPS):
        rsp = await ModelGateway.chat(ctx.model, msgs, tools=ctx.skills.tools, cred=ctx.cred)
        Langfuse.trace_step(node, msgs, rsp)              # 可观测(06)
        if rsp.tool_calls:
            for tc in rsp.tool_calls:
                Guardrails.check_tool_input(tc)            # §5 防线
                out = await McpRuntime.call(tc)            # 工具执行
                out = Guardrails.sanitize(out)             # ★外部内容回流先过滤
                msgs.append(tool_result(tc, out))
        else:
            return Raw(ok=True, content=rsp.content)
    return Raw(ok=False, soft_fail=True)                  # 超步数→可重规划
```

## 4. MCP 集成
```text
AgentRuntime → McpRuntime(Client) → MCP Server(浏览器/文件/DB/搜索/企业API)
```
- 复用现成 MCP Server（含 `browser-pilot` 做互联网检索）；自研工具也包成 MCP Server 统一接入。
- `McpRegistry`：注册可用 server + 其工具清单 → `SkillStore` 的工具型 skill 引用它。

## 5. ★ 安全纵深防御（三道防线）

威胁：抓来的网页内容里藏指令 → 操纵带工具+带凭证的 Agent。

```text
防线1 输入 Guardrails：外部内容进 Agent / 工具回流时 → 注入检测/内容过滤/PII (复用 Guardrails-AI)
防线2 最小权限：SkillLoader 按 agent.authority.allowedTools 过滤工具；越权不加载
防线3 危险动作 gate：node.dangerous=true 的动作(发布/外发/支付/删除) → 不自动 → 审批收件箱(06)
```
- 配合 `05` 出处：工具产出的事实带 sourceUrl/confidence，审核角色可核对。
- `Guardrails` 拦截记审计日志(06)，便于追溯被注入尝试。

## 6. 数据模型（技能相关）
```sql
CREATE TABLE skill (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL, kind TEXT NOT NULL,        -- manual|tool
  trust TEXT NOT NULL DEFAULT 'private',          -- official|verified|community|private
  status TEXT NOT NULL DEFAULT 'draft',           -- draft|published|deprecated
  capability TEXT,                                 -- 关联词表(03)
  owner TEXT, UNIQUE(name));
CREATE TABLE skill_version (
  id UUID PRIMARY KEY, skill_id UUID REFERENCES skill(id),
  version TEXT NOT NULL,
  content TEXT,            -- manual: SKILL.md 正文
  mcp_server TEXT, tool TEXT, io_schema JSONB,   -- tool: MCP 绑定 + schema
  permissions JSONB,       -- 声明会用的工具/权限
  UNIQUE(skill_id, version));
CREATE TABLE skill_invocation (   -- 调用日志(可观测/计费)
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  agent_id UUID, skill_id UUID, latency_ms INT, cost_cents INT,
  status TEXT, created_at TIMESTAMPTZ DEFAULT now());
```

## 7. 执行时序
```text
Temporal Activity run_node
 → Router.select(node.caps) → Agent
 → ContextAssembler: 记忆切片 + 技能(手册/MCP工具) + 模型 + 短时凭证
 → _loop: Model(LiteLLM) ↔ [Guardrails→MCP工具→Guardrails] 多轮
 → ResultEnvelope(出处) → MemoryCenter.write → 返回 Temporal
```

## 8. 边界与错误
- 工具失败 → soft_fail 上报 → 重试/补偿/重规划(03)。
- Guardrails 判定注入 → 阻断该工具调用 + 审计 + 可选升级人审。
- 凭证缺失 → 能力缺口，提示连模型(06)，节点挂起而非静默失败。
- 超 MAX_STEPS → 标 replannable，交 03 有界重规划。
